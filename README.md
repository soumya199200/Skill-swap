# Skill-swap

## 📌 Description

This project is a web-based platform designed for college students to learn and grow without spending money. Users can teach skills to earn credits and use those credits to learn new skills or get career guidance from mentors.

---

## 🚀 Core Idea

The platform follows an **"Earn by Learning"** model:

* Students teach skills → Earn Credits
* Students use credits → Learn new skills
* Students use credits → Get career guidance

---

## 👥 Users

1. Students
2. Mentors (Seniors / Alumni)

---

## 🔁 Workflow

1. User signs up and logs in
2. User adds skills (e.g., Python, UI/UX)
3. User can:

   * Teach a skill → Earn credits
   * Learn a skill → Spend credits
4. User can use credits for:

   * Learning new skills
   * Getting mentorship (resume review, mock interviews)
5. User grows and becomes job-ready

---

## 💰 Credit System

* Teach a session → +50 Credits
* Learn a session → -30 Credits
* Mentorship → -40 Credits

Credits act as an internal currency within the platform.

---

## 🧩 Features

* User Authentication (Login/Signup)
* Skill Exchange System (Teach & Learn)
* Credit Management System
* Mentorship & Career Guidance
